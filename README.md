# JavaScript Counter App

A simple and responsive Counter App built using **HTML**, **CSS**, and **Vanilla JavaScript**. This project was created to practice DOM manipulation, event handling, and keyboard interactions.

## ✨ Features

- ➕ Increment counter
- ➖ Decrement counter (does not go below 0)
- 🔄 Reset counter
- ⌨️ Keyboard shortcuts:
  - `+` → Increment
  - `-` → Decrement
  - `Arrow Up` → Increment
  - `Arrow Down` → Decrement
  - `R` → Reset
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
